# KEE ITS Geospatial Installation Plan

Static GitHub Pages app for the ITS installation plan along:

- Kampala-Entebbe Expressway (KEE)
- Kampala Northern Bypass (KNBP)
- Entebbe Dual Carriageway (EDC)

Recommended GitHub Pages setting:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/docs`

The root `index.html` also redirects to `docs/`, so serving from repository root works too.

Planning note: coordinates are planning-grade and must be confirmed by detailed site survey before procurement, installation or construction.
